# Device bezels

Photographic product bezels from Apple Design Resources
(https://developer.apple.com/design/resources/#product-bezels), included to
composite app screenshots into real device frames — the use these assets are
published for. When using exported images in marketing materials, review
Apple's Marketing Resources and Identity Guidelines:
https://developer.apple.com/app-store/marketing/guidelines/#section-products

Each PNG has a transparent screen cutout; the cutout geometry is cataloged in
`src/lib/device-bezels.ts` (measured from the alpha channel).
