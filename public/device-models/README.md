# Device 3D models

GLB models rendered with three.js for the screenshot editor's "3D" device
style. Ported from the MIT-licensed AppScreen project
(https://github.com/YUZU-Hub/appscreen).

Attribution (CC BY 4.0 — https://creativecommons.org/licenses/by/4.0/):

- **iPhone 15 Pro Max 3D Model** by MajdyModels
  (https://sketchfab.com/majdymodels)
- **Samsung Galaxy S25 Ultra 3D Model** by mistJS
  (https://sketchfab.com/mistjs)

Screen-plane geometry and per-model calibration constants live in
`src/lib/device-models.ts`.
